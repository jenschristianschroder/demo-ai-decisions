# Sample RFP Data — Acme Public Services

This directory contains a sample Request for Proposal (RFP) package from the fictional buyer **Acme Public Services**. The content is used as demo data for the AI Decisions application.

## Contents

| File | Description |
|---|---|
| `acme-public-services-rfp.md` | The full RFP document with all requirements, evaluation criteria, and key dates. |
| `acme-security-questionnaire.md` | Security questionnaire attachment (placeholder for `.docx`). |
| `acme-pricing-template.md` | Pricing template attachment (placeholder for `.xlsx`). |

## Why Markdown?

In a real procurement process these attachments would be distributed as PDF, Word, and Excel files. For this demo the documents are provided as Markdown so they can be easily parsed and processed by the application without requiring binary file conversion. The content and structure mirror what a real RFP package would contain.
